# bin/

Pre-compiled render binaries live here.

| File | Platform |
|------|----------|
| `render-linux` | Linux x64 |
| `render-mac` | macOS Intel |
| `render-mac-arm` | macOS Apple Silicon (M1/M2/M3) |
| `render-windows.exe` | Windows x64 |

Binaries are built automatically by the `release.yml` workflow when a version tag is pushed.
To build manually, see the root README.

> **Instance owners:** pull from upstream to get the latest binary.
> You should never need to build this yourself.
